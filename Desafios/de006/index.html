Tags em HTML5

Em HTML5, podemos inserir elementos em uma página para a Web usando as Tags. Abaixo temos uma lista de elementos e se você passar o mouse em cima verá qual a Tag usada para adicioná-lo:

  • Parágrafos
  • Títulos e sub-títulos
  • Imagens
  • Códigos-fonte
  • Tabelas enumeradas
  • Tabelas com demarcadores
  • hiperlinks
  • Destaque ao texto
  • Ênfase ao texto
  • Quebra de linha
  • Abreviações
